# Assets

Esta pasta contém assets estáticos do projeto.

## Logo CIAS

Para o template de exportação do TRP funcionar corretamente, adicione o arquivo:

- **cias-logo.png** - Logo do CIAS (altura recomendada: ~42px, formato PNG com fundo transparente)

O componente `TrpExportTemplate` tentará carregar este logo automaticamente. Se o arquivo não existir, será exibido um placeholder com o texto "CIAS".
